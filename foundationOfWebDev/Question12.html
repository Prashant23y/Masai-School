FrontEnd Development

HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

JavaScript

JavaScript is a versatile, lightweight, interpreted programming language primarily used to add interactivity to web pages. It can also be used in other environments like Node.js for server-side development. Key Features : Dynamic Typing: Variable types are checked during runtime. Interpreted Language: Code is executed line by line. Single-Threaded: Executes one task at a time. Event-Driven: Designed to handle events, like user interactions. Cross-platform: Runs on various operating systems and browsers
